# Streaming: ABR, HLS, DASH

Streaming platforms adjust video quality dynamically.

---

## ABR (Adaptive Bitrate Streaming)
Adjusts quality based on:
- Network speed
- Buffer health
- Device capability

---

## HLS
Apple standard using segmented video files.

---

## DASH
Open standard alternative to HLS.

---

## Streaming Workflow
Client → Manifest → Video Segments → Buffer → Playback

---

## Why Streaming Works Smoothly
Segments allow switching quality without restarting playback.
