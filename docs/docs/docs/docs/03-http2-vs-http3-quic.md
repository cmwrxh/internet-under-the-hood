# HTTP/2 vs HTTP/3 (QUIC)

## HTTP/2 Improvements
- Multiplexing
- Header compression
- Server push

Runs over TCP.

---

## HTTP/3 Improvements
Runs over QUIC (UDP-based).

Benefits:
- Reduced connection setup time
- Better mobile performance
- Faster reconnections

---

## Why QUIC Matters
TCP requires full handshake recovery after packet loss.  
QUIC recovers faster.
