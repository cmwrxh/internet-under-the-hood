# TLS Handshake

TLS ensures secure encrypted communication.

---

## Handshake Steps

1. Client Hello
   - Supported cipher suites
   - Random values
   - TLS version

2. Server Hello
   - Selected cipher suite
   - Certificate

3. Key Exchange
4. Session Keys Established
5. Encrypted Communication Begins

---

## TLS Versions
- TLS 1.2
- TLS 1.3 (faster & more secure)
