# DNS and CDN

## DNS Overview
Domain Name System translates human-readable domains into IP addresses.

Example:

google.com → 142.250.190.78

### DNS Query Flow
Client → Recursive Resolver → Root → TLD → Authoritative Server

---

## CDN Overview
Content Delivery Networks improve speed and reliability by caching content closer to users.

Examples:
- Cloudflare
- Akamai
- AWS CloudFront

---

## CDN Request Flow
User → Edge Node → Origin Server

Benefits:
- Reduced latency
- Load balancing
- DDoS protection
